# Decidon
### Prosopographical Directory of the French Third Republic

Decidon is a web application providing a searchable prosopographical directory of French parliamentarians and government members under the Third Republic (1870вЂ“1940).

---

## 1. Sources

### Primary sources

- [Sycomore](https://www2.assemblee-nationale.fr/sycomore/liste/) вЂ” National Assembly deputies database
- [Senate website archive](https://www.senat.fr/senateurs-3eme-republique/senatl.html) вЂ” Third Republic senators

### Additional sources

- [Wikipedia](https://www.wikipedia.org/)
- [Wikidata](https://www.wikidata.org)

## 4. Prerequisites

- Python 3.10 or higher
- pip

## 5. Installation

**Step 1 вЂ” Create and activate a virtual environment**

```bash
python3 -m venv .venv
```

On macOS / Linux:
```bash
source .venv/bin/activate
```

On Windows:
```bash
.\.venv\Scripts\activate
```

**Step 2 вЂ” Install dependencies**

```bash
pip install -r requirements.txt
```

---

## 6. Database initialisation

The database is generated from a TSV source file using the migration script.

**Step 1 вЂ” Place the source file**

Store `prosopography.tsv` at the following path:

```
./db/data/prosopography.tsv
```

**Step 2 вЂ” Run the migration**

```bash
python3 db/run_migration.py
```

This creates `prosopography.db` at `./db/data/prosopography.db`.

---

## 7. Launch the app

Run the following command from the project root:

```bash
uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

The application will be available at [http://127.0.0.1:8000](http://127.0.0.1:8000).

### Main endpoints

| Method | Endpoint                          | Description                              |
|--------|-----------------------------------|------------------------------------------|
| GET    | `/persons`                        | List or search persons                   |
| GET    | `/persons/{person_id}`            | Get a person by ID                       |
| GET    | `/persons/grouped-mandates`       | Search persons with their grouped mandates |
| GET    | `/persons/by-role`                | Get persons holding a role at a given date |
| GET    | `/legislatures`                   | List legislatures                        |
| GET    | `/legislatures/{id}/members`      | Get members of a legislature             |
| GET    | `/lookup`                         | Fuzzy name lookup                        |
